# Frontend Go Menu

Sistema de cardápio - Frontend

## Instalação

```bash
npm install
```

## Rodar o projeto

```bash
npm run dev
```

O frontend estará disponível em: http://localhost:4000

## Páginas

- `/login` - Login do usuário
- `/cadastro` - Cadastro do usuário
- `/restaurantes` - Seleção e gerenciamento de restaurantes
- `/dashboard` - Painel administrativo do restaurante selecionado
- `/cardapio/:slug` - Cardápio público (via link único)
